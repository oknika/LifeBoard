# LifeBoard

> Your personal dashboard for everyday life.

LifeBoard is an open-source desktop application that helps organize different aspects of daily life in one place.

The project started from a simple idea:

Instead of using multiple applications for health records, personal notes, finances, and daily tasks, why not have one application that feels like a personal companion?

LifeBoard is designed to grow module by module, with every feature solving a real problem from everyday life.

---

## ✨ Vision

Build a desktop application that I genuinely use every day.

Every feature is added because it improves my own daily workflow, making LifeBoard both a practical personal tool and a showcase of modern desktop application development.

---

## 🗺 Roadmap

### Version 0.1 — Health Journal

The first module focuses on personal health tracking.

Planned features:

- Record blood pressure
- Record pulse
- Record body weight
- Add personal notes

Future versions will expand LifeBoard with additional modules.
