# LifeBoard Roadmap

LifeBoard is built incrementally.

Every release should solve a real problem and become part of my daily life before moving on to the next milestone.

---

# Version 0.1 — Health Journal

Goal:
Record personal health information in a simple and reliable way.

Features:

- [ ] Record blood pressure
- [ ] Record pulse
- [ ] Record body weight
- [ ] Add notes
- [ ] View measurement history

Status:
🚧 In Planning

---

# Version 0.2 — Personal Notes

Goal:
Capture ideas and important information in one place.

Features:

- [ ] Create notes
- [ ] Edit notes
- [ ] Search notes
- [ ] Organize with tags

---

# Version 0.3 — Personal Finance

Goal:
Track income and expenses.

Features:

- [ ] Income
- [ ] Expenses
- [ ] Categories
- [ ] Monthly summary

---

# Version 0.4 — Tasks

Goal:
Manage everyday tasks.

Features:

- [ ] To-do list
- [ ] Due dates
- [ ] Completed tasks

---

# Version 0.5 — Synchronization

Goal:
Synchronize data with the remote server.

Features:

- [ ] User authentication
- [ ] Upload changes
- [ ] Download changes
- [ ] Conflict resolution

---

# Version 1.0

LifeBoard becomes my primary personal dashboard.

Modules:

- Health Journal
- Notes
- Finance
- Tasks

All modules work offline and support synchronization.
